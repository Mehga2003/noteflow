# NoteFlow 📝

NoteFlow is a Smart Notes Management System built using Django.

It allows users to:
- Create notes
- Edit notes
- Delete notes
- Search notes
- Securely login/logout
- Manage personal notes with a modern UI

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone <repository-link>
```

## 2. Move into Project Folder

```bash
cd smartnotes
```

## 3. Install Django

```bash
pip install django
```

## 4. Run Migrations

```bash
python manage.py migrate
```

## 5. Create Superuser

```bash
python manage.py createsuperuser
```

## 6. Start Server

```bash
python manage.py runserver
```
